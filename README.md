# DUNGEON BREAK REBORN — Phase 5

ブラウザで遊べるローグライト・オートバトルアクションゲームです。GitHub Pagesでそのまま公開できる静的ファイル構成です。

## 起動方法

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。GitHub Pagesでは、このフォルダの中身をリポジトリ直下に配置してください。

## Phase 5 追加内容

- スキル進化システム
  - 条件を満たすと自動的に進化スキルが解放されます。
  - 火、雷、毒、氷、召喚、Abyssal系の進化を追加しました。
- 図鑑 / 進化画面
  - タイトルから進化条件と装備セット効果を確認できます。
- 深淵の裂け目イベント
  - 4Waveごとに一時イベントが出現します。
  - 装備、経験値、回復、火力、報酬上昇などから選択できます。
- 装備セット
  - 装備にセットタグを追加しました。
  - 2部位、4部位で追加効果が発動します。
- 装備管理改善
  - 装備ロックを追加しました。
  - ロック中の装備は分解対象から外れます。
  - Common/Rare一括分解もロックを尊重します。
- 戦闘効果追加
  - 進化スキルと装備セットが、火炎円環、連鎖雷、毒雲、氷晶ノヴァ、使い魔、黒穴などに反映されます。

## 保存キー

```text
dungeonBreakReborn.phase5.save
```

Phase 4以前の保存がある場合は、初回読み込み時に引き継ぎます。
